Ok, now what?

A CLASSHas properties and functions which can be given visibility. There are 3 types:

  • Private (visible only within the class)
  • Protected (accessible from within the defining class and its ancestor)
  • Public. You got it, EVERYWHERE

FROM WITHIN A FUNCTION we can access a class properties by prepending the name with $this->

class Person { protected $name; public function __construct($name) { $this->name = $name; } public function getIntroduction() { return sprintf( __('My name is %s'), $this->name ); } }
Above: the "%s" refers to PHP returning a string;

A class is instantiated through the NEW keyword; After that, the properties and functions can be accessed.

$person = new Person('Pedro'); echo $person->getIntroduction(); // This prints "My name is Pedro"

PHP __construct

What is it?

If you create a __construct() function, PHP will automatically call this function when you create an object from a class.

So, it saves us from calling the set_name() method that would have had to be used in the top

REVIEW:
  • getters, setters: these are older methods used in PHP to GET and SET info for a class
  • __construct: calls the function automatically when an object is created, preventing the need for SETTERS in a class
abstractmethods

Abstract methods MUST be implemented by an inheriting class.

ONLY the class IMPLEMENTING the abstract method can be instantiated:

abstract class Person { abstract public function getName(); public function getIntroduction() { return sprintf( __('My name is %s'), $this->getName() ); } } // Person cannot be instantiated class Manuel extends Person { public function getName() { return 'Manuel'; } } // Manuel can be instantiated $manuel = new Manuel();

STATIC methods

Static lives under the class, and not under an INSTANTIATION

Use self:: from within the class, and class + ::outside of the class to access a STATIC method

class Factory { protected static $instances = []; public static function registerInstance($handle, $instance) { self::$instances[$handle] = $instance; } public static function getInstance($handle) { return self::$instances[$handle]; } } $engine = Factory::getInstance('Engine');
